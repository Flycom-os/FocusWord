# 🚀 Быстрый Старт: Слайдеры и Страницы

## За 5 минут до первого слайдера на странице

### Шаг 1: Подготовите Медиа (1 минута)

```
Dashboard → Администратор → Медиафайлы
```
- Загрузите 3-5 изображений для слайдов
- Добавьте альтекст (например: "Баннер - Сезонная коллекция")

### Шаг 2: Создайте Слайдер (1.5 минуты)

```
Dashboard → Администратор → Слайдеры
```

**Нажмите:** "Добавить слайдер"

Заполните форму:
```
Название: Главный баннер
Slug: main-banner
Описание: Основной слайдер главной страницы
```

**Сохраните** → Слайдер создан ✅

### Шаг 3: Добавьте Слайды (2 минуты)

В таблице слайдеров найдите созданный слайдер и нажмите на него.

**Нажмите:** "Добавить слайд" для каждого изображения

Для каждого слайда:
```
Название: Весенняя коллекция 2026
Описание: Новые модели и дизайны. Используйте скидку 30%
Ссылка: https://yoursite.com/spring-collection
Порядок: 0, 1, 2 (порядок отображения)
Медиа: Нажмите "Выбрать медиа" → выберите изображение
```

**Сохраните** каждый слайд → Слайды добавлены ✅

### Шаг 4: Создайте Страницу (0.5 минуты)

```
Dashboard → Администратор → Страницы
```

**Нажмите:** "Создать страницу"

Основные поля:
```
Название: О нас
Slug: about-us
Содержимое: <h2>Добро пожаловать</h2><p>Текст о компании...</p>
```

**В боковой панели:**
- **Основной слайдер:** Выберите "Главный баннер" из списка
- **Статус:** "Опубликовано"

**Сохраните** → Страница создана ✅

### Шаг 5: Просмотрите Результат (0 минут)

```
Откройте в браузере: https://yoursite.com/pages/about-us
```

Вы должны увидеть:
1. Заголовок "О нас"
2. **Слайдер с автопроигрыванием** ← 🎉 Это работает!
3. Текст "Добро пожаловать..."

## 🎮 Управление Слайдером на Странице

Пользователь видит:

| Элемент | Функция |
|---------|---------|
| **← →** Стрелки | Ручная навигация между слайдами |
| **● ● ●** Точки | Быстрый переход на нужный слайд |
| Автопроигрывание | Каждые 5 секунд переход на следующий слайд |

## 🐛 Решение Проблем

### Слайдер не отображается на странице
- ✅ Проверьте что слайдер выбран в поле "Основной слайдер"
- ✅ Проверьте что слайдер имеет минимум 1 слайд
- ✅ Проверьте что слайды имеют связанные медиа файлы

### Изображения не видны в слайдах
- ✅ Убедитесь что медиа загружены (зелёная галочка в медиафайлах)
- ✅ Проверьте что альтекст добавлен к медиа
- ✅ Обновите страницу (F5 или Ctrl+R)

### Слайдер не автопроигрывается
- ✅ Это нормально - включится после 1-го взаимодействия пользователя
- ✅ Настройку можно изменить в коде компонента

## 🎯 Что Дальше?

После того как вы создали свой первый слайдер:

1. **Создайте больше слайдеров** для разных разделов
2. **Используйте разные стили** - создавайте индивидуальные слайдеры
3. **Добавьте ссылки** на важные страницы и категории
4. **Кастомизируйте стили** в `PageSlider.module.css`
5. **Интегрируйте в разные страницы** - каждая может иметь свой слайдер
